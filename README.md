# ROBLOX-HACKPAD
---
This is a hackpad I have made for my gaming purposes, it could be used for anything however

it features 9 keys and a joystick meant to be moved with your thumb, it also has a little screen on the top of the macropad
it is plugged in using a usb c into the xiao rp2040 on the top right of the case, you can of course change what the keys and joystick do
the screen is plugged into the connectors, and the same goes for the joystick
